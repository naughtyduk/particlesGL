ASCII Effect Library

Interactive Three.js Particle System with Mouse Glitch Effects

Live Demo

Move your mouse over the canvas below to see the interactive ASCII logo effect in action. The particles will respond to your mouse movement with a glitch effect and smooth rotations.

How to interact:

  • Move your mouse over the canvas to see the logo rotate
  • The particles will scatter and glitch around your mouse cursor
  • Move your mouse quickly for more dramatic effects
  • The animation pauses when not in view for better performance

Usage Example

HTML Setup:

<!-- Include Three.js -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>

<!-- Create target element -->
<div id="ascii-logo-insert"></div>

<!-- Include the ASCII Effect library -->
<script src="your-ascii-effect-library.js"></script>

JavaScript Initialization:

// Initialize the effect
ASCIIEffect.init();

// Clean up when needed
ASCIIEffect.cleanup();
